## API overview

The NeetoRecord API is organized around [REST](https://en.wikipedia.org/wiki/REST) principles and uses standard [HTTP](https://en.wikipedia.org/wiki/HTTP) response codes. All API endpoints return [JSON](https://en.wikipedia.org/wiki/JSON) responses and require authentication via API keys.

Before you start with NeetoRecord APIs you will need the following:

1. [A NeetoRecord workspace](/getting-started/identifying-subdomain).
2. [An API key](/getting-started/authentication).

## Quick start

Get started with the NeetoRecord API in minutes. Make your first API call and retrieve your recordings. You should have your [API key](/getting-started/authentication) & [subdomain](/getting-started/identifying-subdomain) ready with you.

### Step 1: Make Your First API Call

Let's retrieve all recordings in your workspace using a [GET request](https://en.wikipedia.org/wiki/HTTP#Request_methods):

<CodeGroup>
```bash curl
curl -X GET "https://{your-subdomain}.neetorecord.com/api/external/v2/recordings" \
  -H 'X-Api-Key: your-api-key' \
  -H 'Content-Type: application/json'
```

```javascript JavaScript
const response = await fetch(
  "https://{your-subdomain}.neetorecord.com/api/external/v2/recordings",
  {
    method: "GET",
    headers: {
      "X-Api-Key": "your-api-key",
      "Content-Type": "application/json",
    },
  }
);

const data = await response.json();
console.log(data);
```

```python Python
import requests

url = "https://{your-subdomain}.neetorecord.com/api/external/v2/recordings"
headers = {
    "X-Api-Key": "your-api-key",
    "Content-Type": "application/json"
}

response = requests.get(url, headers=headers)
data = response.json()
print(data)
```

```php PHP
<?php
$curl = curl_init();

curl_setopt_array($curl, [
    CURLOPT_URL => "https://{your-subdomain}.neetorecord.com/api/external/v2/recordings",
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_HTTPHEADER => [
        "X-Api-Key: your-api-key",
        "Content-Type: application/json"
    ],
]);

$response = curl_exec($curl);
curl_close($curl);

$data = json_decode($response, true);
print_r($data);
?>
```

</CodeGroup>

<Warning>
  Replace `your-subdomain` with your actual workspace subdomain and
  `your-api-key` with your generated API key.
</Warning>
